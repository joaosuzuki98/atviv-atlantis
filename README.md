# Executando o projeto

## Instale o pnpm

Este projeto faz uso do gerenciador pnpm, se não o tiver, instale-o:

```shell
npm i -g pnpm
```

## Instale as dependências

```shell
pnpm install
```

## Rodando o projeto

```shell
pnpm dev
```

Acesse o site pelo endereço [localhost:3000](http://localhost:3000)
